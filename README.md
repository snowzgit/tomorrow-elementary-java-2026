# Tic-Tac-Toe

A Java implementation of Tic-Tac-Toe supporting local player-versus-player
and player-versus-computer matches.

## Overview

This repository contains the final project developed for Tomorrow's in-person
Elementary Java course, held at the Federal University of Bahia (UFBA) from
July 13 to 24, 2026.

The project consists of a Tic-Tac-Toe game with two game modes:
local player versus player and player versus computer. In
player-versus-computer mode, the user can choose between easy and medium
difficulty levels.

## Features

* Local player-versus-player gameplay
* Player-versus-computer gameplay
* Easy and medium computer difficulty modes
* Turn-based game flow
* Detection of wins, losses, and draws
* Validation of player moves

## Technologies

* Java 8 or later

## Requirements

To compile and run the project, you must have the following installed:

* Java Development Kit (JDK) 8 or later
* Git, if cloning the repository from the command line

## Installation

Clone the repository:

```bash
git clone https://github.com/snowzgit/tomorrow-elementary-java-2026.git
````

Enter the project directory:

```bash
cd tomorrow-elementary-java-2026
```

## Usage

Compile and run the `Main` class:

```bash
javac <path-to-main-class>.java
java <fully-qualified-main-class>
```

Alternatively, open the project in a Java-compatible IDE and run the `Main`
class directly.

After starting the game, select one of the available game modes:

* Player versus player
* Player versus computer

  * Easy difficulty
  * Medium difficulty
